# Saoirineu Palette

Base palette derived from the Santo Daime flag and the symbols the project should foreground.

| Token | Hex | Use |
| --- | --- | --- |
| `brand-white` | `#f7f4ea` | Main light surface, warm white from the upper band |
| `brand-blue` | `#3f84c2` | Primary brand blue, middle band, links, theme color |
| `brand-blue-deep` | `#2a628f` | Stronger blue for buttons, active nav, outlines |
| `brand-green` | `#0f6f57` | Secondary accent, lower band, confirmations, CTA hover |
| `brand-green-deep` | `#0b5a46` | Darker green for pressed states |
| `brand-gold` | `#e8c24c` | Moon, star, sacred-symbol accent |
| `brand-gold-soft` | `#f3df8c` | Highlights and soft glow |
| `brand-ink` | `#24364d` | Main text and structural contrast |
| `brand-sand` | `#e5dcc7` | Borders and muted separators |
| `brand-mist` | `#edf3f6` | Page background transition color |

Symbol hierarchy used in the assets:

- `favicon`: flag bands plus crescent, kept intentionally simple for browser-tab legibility.
- `site mark`: flag bands plus crescent and six-point star.
- `app icon/logo`: flag bands plus crescent, six-point star, and Caravaca cross.
