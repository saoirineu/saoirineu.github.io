# Pod Frontend

Escopo: aplicação client-side que autentica via Firebase e exibe os dados RDF (grafos e texto).

- O código vive em `frontend/`.
- Escolha framework moderno (React, Vue, Svelte, etc.) conforme a funcionalidade, sempre buscando a menor solução possível.
- Registre dependências e comandos essenciais aqui para facilitar a revisão pelo pod mestre.
- Consulte `docs/pods.md` para princípios e regras de convivência entre pods.
